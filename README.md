# Express Note Taker
### GitHub: [ treavorwills ]( https://github.com/treavorwills )
## Description
Web application to write and save notes using an Express.js backend and data stored in a JSON file

## How it's made
### Languages:
- JavaScript ES6+
- CSS3
- HTML5

### Tools: 
- Express
- Node 
- Bootstrap
